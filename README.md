# Cardápio Digital

Este é um projeto didático criado durante um curso de Desenvolvimento Fullstack, após o módulo de HTML e CSS. O objetivo é praticar conceitos fundamentais de criação de interfaces web estáticas, representando um cardápio digital para um restaurante focado em hambúrgueres, acompanhamentos e bebidas.

## Tecnologias Utilizadas

- **HTML**: Para a estrutura semântica da página, incluindo cabeçalho, navegação e seções de produtos.
- **CSS**: Para estilização visual, layout responsivo e aplicação de cores definidas em `RefCores.txt`.
- **Imagens**: Arquivos de imagem compactados para ilustrar os produtos.
- **Arquivos de Texto**: `RoteiroTexto.txt` para descrições e roteiro de navegação, e `RefCores.txt` para referências de cores.

## Como Usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/vinicius-benedito/cardapioOnline.git
   ```

2. Navegue até a pasta do projeto:
   ```bash
   cd cardapioOnline
   ```

3. Abra o arquivo `cardapio.html` em um navegador web para visualizar o cardápio.

Não há dependências externas ou servidor necessário, pois é um site estático.

## Estrutura do Projeto

- `cardapio.html`: Arquivo principal da página.
- `cardapio.css`: Estilos CSS.
- `imagensCompactadas/`: Pasta com imagens e arquivos de referência.
  - `imagens/`: Imagens dos produtos.
  - `requisitos/`: Arquivos de texto com roteiro e cores.
